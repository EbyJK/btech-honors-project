Urban Vulnerability Assessment  using satellite images. 

Mini Project associated with B.tech Honors. 

5 ml models:
1. Vegetation
2. Roads
3. Slums
4. Built up Area
5. Water

5 models are trained using Unet and deeplab models.
5 models will result 5 layers which will be superimposed in the final output and the regions with vulnerability risk is mapped and clustered on pixel level. The result is rendered on the frontend.

3 categories of risk. High/Medium/Low
